# svelte5-state prototype

Spike to validate module-level `$state` sharing and document the `$derived` export
limitation/workaround in Svelte 5 runes. See [VERDICT.md](./VERDICT.md) for results.

## Setup

```sh
pnpm install
```

## Developing

```sh
pnpm dev
# or open in browser automatically
pnpm dev -- --open
```

## Building

```sh
pnpm build
```

Preview the static build:

```sh
pnpm preview
```
