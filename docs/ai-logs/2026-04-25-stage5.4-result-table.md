## Technical Details

### Svelte 5 Event Handler Syntax
Fixed deprecation warnings by using modern syntax:
```svelte
<!-- Before (deprecated) -->
on:focus={(e) => handler(e)}
on:keydown={(e) => handler(e)}

<!-- After (Svelte 5) -->
onfocus={(e) => handler(e)}
onkeydown={(e) => handler(e)}
```

### CalculatorState Integration
- `CalculatorState` manages both input rows and calculation results
- Triggers recalculation when entity selection or energy values change
- Results cached in `Map<normalizedMevNucl, {stpCekMuM, csdaRangeCm}>` for performance

### Unit Scaling Logic

`autoScaleLengthCm()` lives in `src/lib/utils/unit-conversions.ts` and supports
five SI prefixes (nm, µm, mm, cm, m) so the displayed CSDA range stays in a
human-readable range. See `docs/progress/stage-5.4-result-table.md` for the
full implementation.
