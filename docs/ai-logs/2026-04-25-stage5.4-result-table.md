### Task 2: Calculator state ‚Äî `calculator.svelte.ts`
- **Status**: completed
- **Stage**: 5.4
- **Files changed**:

    `createCalculatorState(entitySelection, service)` factory; `CalculatedRow`

    `stoppingPower`, `csdaRangeCm`

    `vi.useFakeTimers()` to control 300ms debounce
- **Decision (key)**: results cached in a `Map<normalizedMevNucl, {stpCevUm, csdaRangeCm}>`

  detect mutations.

  calculation to force Svelte to re-read the Map in derived getters. ESLint
  `no-unused-vars` false-positives on this variable (it IS used on line 223);
  suppressed with `// eslint-disable-next-line`.
- **Decision**: `stpDisplayUnit` is a plain computed function (not `$derived`) because
  it reads from `entitySelection.selectedMaterial` which is already reactive.
- **Commit**: `147f989 feat(stage5.4): calculator state with debounced calculation`
