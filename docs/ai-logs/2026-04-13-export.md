- **Decisions**:
  - STP column moved last because CSDA Range (structural result) is more
    directly comparable to "how far does the particle travel" — the most
    common non-specialist question. STP (energy-loss rate per unit length)
    is the derived quantity researchers typically look up last.
  - Share URL is the primary sharing action — placing it rightmost makes it
    the visual anchor for the sharing cluster.
  - CSV modal in advanced mode only: basic-mode users want a quick download;
    advanced users are doing data analysis and care about separator and
    encoding compatibility with their tools (pandas, R, Excel locale).
  - PNG advanced-mode only: PNG is a convenience raster capture. SVG is the
    archival format and is useful even in basic mode. Keeping PNG
    advanced-mode-only reduces the basic mode UI surface.
  - BUILD at the very bottom: it is build provenance, not physics data. A
    reader skimming the metadata block cares first about what particle,
    material, and programs were used; the commit hash is a debug detail.
  - `Page n / N` on single-page basic PDFs as well as multi-page advanced
    PDFs: keeps the format consistent and tells the reader there is only one
    page without them having to wonder.
- **Issues**: None.
