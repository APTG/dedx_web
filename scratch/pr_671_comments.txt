## Pull request overview

This PR adds a stopping-power output unit selector to the Advanced calculator and unifies calculator/plot URL state around the shared `sunit=` parameter, while moving single-entity STP conversion to display time.

**Changes:**
- Adds shared STP unit codec/state plus render-time mass-to-display unit conversion.
- Adds calculator STP header menus for single-result and multi-program result headers, including mobile sheet behavior.
- Updates URL handling, documentation, and unit/component/E2E coverage for `sunit=` and legacy plot `stp_unit=` fallback.
